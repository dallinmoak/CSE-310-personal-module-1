# Sprint1 Modules - GIS Mapping

A demo of GIS mapping using [ArcGIS](https://developers.arcgis.com/)

## Instructions for Build and Use

Steps to build and/or run the software:

1. visit the prod site at [cse-310-personal-module-1.vercel.app](https://app.vercel.com](https://cse-310-personal-module-1.vercel.app/)

Instructions for using the software:

1. look at the map, ineract with the map using standard web interactions.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

1. install nodejs/npm (LTS or later)
2. install dependencies with `npm i`
3. run the dev command `npm run dev`
